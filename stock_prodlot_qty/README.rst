======================
Stock Prodlot Quantity
======================

Simple module that displays available qty in prodlot name if context
key is set.  Useful for Many2one dropdowns.

Installation
============

There are no special installation instructions for this module.

Configuration
=============

When defining a view or requesting a prodlot name set a context key
"show_qty" to True.

Usage
=====

None
